# VR恐竜博物館

福井県立恐竜博物館の3Dモデルとアニメーションするティラノサウルスを楽しめる、ウェブベースのVR/AR体験です。

## デモ

*   **博物館外観:** [https://code4fukui.github.io/vr-dinosaur-museum/](https://code4fukui.github.io/vr-dinosaur-museum/)
*   **ティラノサウルスビューアー:** [https://code4fukui.github.io/vr-dinosaur-museum/trex.html](https://code4fukui.github.io/vr-dinosaur-museum/trex.html)

## 概要

このプロジェクトは[A-Frame](https://aframe.io/)を使用して構築されており、2つのバーチャル体験を提供します。

1.  **博物館ツアー:** 福井県立恐竜博物館の外観をバーチャルで探索できます。宙に浮いている赤い矢印をクリックすると、別の視点へ移動します。
2.  **ティラノサウルスビューアー:** ティラノサウルスのモデルを操作できるインタラクティブな3Dビューアーです。このデモには、対応するiOSデバイス上で恐竜を拡張現実（AR）で表示するためのリンクが含まれています。

## 主な機能

*   福井県立恐竜博物館の建物のバーチャルツアー
*   自動回転するインタラクティブな3Dティラノサウルスモデルのビューアー
*   A-Frameを活用したWebVR体験
*   博物館シーンにおけるポイント＆クリック方式のナビゲーションシステム
*   iOS Quick Look用のUSDZファイルによるティラノサウルスモデルのAR対応

## 操作方法

*   **マウス:** クリック＆ドラッグで両シーンの周囲を見渡すことができます。
*   **博物館シーン:** 宙に浮いている赤い矢印をクリックすると、新しい場所に移動します。
*   **ティラノサウルスシーン (iOS):** 「ARで見る on iOS」リンクをクリックすると、実際の環境にモデルを配置して表示できます。

## 主なアセット

*   `fukui-dinosaur-museum.glb`: 博物館の建物の3Dモデル
*   `dinosaur-trex.glb`: ティラノサウルスの3Dモデル
*   `dinosaur-trex.usdz`: AR対応のティラノサウルスモデル
*   `arrow.glb`: 博物館シーンで使用されるナビゲーション用の矢印

## クレジット

本プロジェクトは[福井県立恐竜博物館](https://www.dinosaur.pref.fukui.jp/)をベースにしています。
