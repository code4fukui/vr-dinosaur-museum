# VR Dinosaur Museum

> 日本語のREADMEはこちらです: [README.ja.md](README.ja.md)

A web-based VR/AR experience featuring a 3D model of the Fukui Prefectural Dinosaur Museum and an animated T-Rex.

## Demos

*   **Museum Exterior:** [https://code4fukui.github.io/vr-dinosaur-museum/](https://code4fukui.github.io/vr-dinosaur-museum/)
*   **T-Rex Viewer:** [https://code4fukui.github.io/vr-dinosaur-museum/trex.html](https://code4fukui.github.io/vr-dinosaur-museum/trex.html)

## Description

This project, built with [A-Frame](https://aframe.io/), provides two virtual experiences:

1.  **Museum Tour:** A virtual exploration of the exterior of the Fukui Prefectural Dinosaur Museum. Navigate between different viewpoints by clicking the floating red arrows.
2.  **T-Rex Viewer:** An interactive 3D viewer for a T-Rex model. This demo includes a link to view the dinosaur in Augmented Reality (AR) on supported iOS devices.

## Features

*   Virtual tour of the Fukui Prefectural Dinosaur Museum building.
*   Interactive 3D T-Rex model viewer with automatic rotation.
*   WebVR experience powered by A-Frame.
*   Point-and-click navigation system in the museum scene.
*   AR support for the T-Rex model via USDZ for iOS Quick Look.

## Controls

*   **Mouse:** Click and drag to look around in both scenes.
*   **Museum Scene:** Click the floating red arrows to move to a new location.
*   **T-Rex Scene (iOS):** Click the "ARで見る on iOS" link to view the model in your environment.

## Key Assets

*   `fukui-dinosaur-museum.glb`: 3D model of the museum building.
*   `dinosaur-trex.glb`: 3D model of the T-Rex.
*   `dinosaur-trex.usdz`: AR-compatible version of the T-Rex model.
*   `arrow.glb`: Navigation arrow used in the museum scene.
